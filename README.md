# solitaire-workspace

Workspace for a TypeScript Solitaire (draw-3) core library and CLI.

## Packages

- `core/` - publishable `@caheinz2/solitaire-core` library
- `cli/` - private terminal app

## Scripts

- `npm test` - run workspace tests
- `npm run test:core` - run core unit tests
- `npm run build` - build all workspaces
- `npm run build:core` - build the core library
